//**🦖 Dino Game (Python)**//
A simple recreation of the classic Chrome Dino Game built with Python and Pygame.
The player controls a dinosaur that must jump over incoming obstacles. The game gradually becomes more challenging as the obstacle speed increases over time.

**Features**
•	Jump mechanic using gravity and velocity
•	Collision detection
•	Increasing difficulty as the score increases
•	Live score display
•	Simple and lightweight implementation using Pygame

**Controls**
Key	Action
Space	Jump

**Requirements**
•	Python 3.x
•	Pygame
Install Pygame with:
pip install pygame

**Running the Game**
Clone the repository:
git clone https://github.com/lefrenchtopper/DINO-game.git
Move into the project folder:
cd DINO-game
Run the game:
python dino_game.py
